# Team Rule Book

## Base

- 슬랙 자주 잘 확인한다!
  - 최대 2주 내에 확인한다

## Meetings

- 정기 회의
  - 월요일 소개원실 랩시간에 대면 회의
- 데일리 스크럼
  - 11시~17시 사이에 슬랙 데일리 스크럼 봇에 댓글로 어제 한 일과 오늘 할 일을 올린다
- 회의는 모두 스누메일로 구글 캘린더 인비 보내서 처리한다

## Git Convention

- 따로 Fork 없이 각자 이름 안 겹치게 브랜치 분기해서 진행
- GitHub flow (main 브랜치 하나만 두고, PR로만 머지 가능)
